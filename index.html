ОБО МНЕ

Пишу приложения на JavaScript, с использованием стандарта ES 6, есть базовые знания Webpack и React/Redux. Владею версткой, с использованием HTML/СSS, а также Jade и Bootstrap. Работала с Web Audio API для создания веб-аудиоплеера, а также с шаблонизатором Jekyll. Хорошие знания Сanvas, его изучила при написании веб-игры (с его помощью была реализована генерация юнитов из деталей). Посмотреть мои работы можно также в профиле на GitHub.

Образование

В 2015 году окончила Белорусский госдарственный технологический университет по специальности редактор-технолог. Долгое время изучала верстку самостоятельно, на сайте htmlAcademy.ru. В 2017 году прошла курсы TeachMeSkills по направлениям "HTML/CSS" и "JavaScript/Jquery". С февраля по август 2018 года проходила курсы RollingScopesSchool. С сентября 2018 года устроилась на работу Frontend разработчиком, занимаюсь версткой и рефакторингом сайта, использую шаблонизатор Jekyll.
